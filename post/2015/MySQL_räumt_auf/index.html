MySQL räumt auf

Jede neue MySQL Major-Version bringt neue Features, erklärt aber auch ein paar alte als veraltet (deprecated). Bisher war es üblich, dass die veralteten Features dann in der übernächsten Version aus MySQL entfernt werden.

Mit dem anstehenden Release von MySQL 5.7 zeigt man sich progressiver. Einige Funktionen, die mit MySQL 5.6 deprecated wurden, stehen in 5.7 bereits nicht mehr zur Verfügung. Eine Übersicht gibt der Artikel Removal and Deprecation in MySQL 5.7.
