Das Git-Buch

Leider hat der Verlag Open Source Press zu Ende Dezember 2015 die Verlagstätigkeit eingestellt. Das ist doppelt schade, erstens weil es ein renommierter deutschsprachiger Verlag mit vielen interessanten Titel war, und zweitens, weil er als erster Verlag einen wirklich modernen crossmedialen Publikationsprozess auf Basis von AsciiDoc etabliert hatte.

Die Autoren haben sämtliche Rechte an den veröffentlichten Texten zurück übertragen bekommen, was eine faire Geste und alles andere als selbstverständlich ist.

Die Autoren des Buches Git - Verteilte Versionsverwaltung für Code und Dokumente haben sich nun entschieden, ihr Werk unter einer CreativeCommons Lizenz zur Verfügung zu stellen. Auf ihrer Website findet man das Buch einschließlich der Quellen in AsciiDoc sowie die Vortragsfolien ihrer Git-Schulungen.
