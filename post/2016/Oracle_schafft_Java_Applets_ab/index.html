Oracle schafft Java Applets ab

Java Applets sind die wohl älteste Plugin-Technologie in der Geschichte des WWW. Während es in den 90er Jahren durchaus sinnvolle Anwendungen dafür gab, sind sie heute aus dem offenen Web weitgehend verschwunden – leider oft abgelöst duch Flash Applets. In Unternehmensanwendungen halten sie sich dagegen weiterhin hartnäckig.

Nachdem Adobe vor einigen Wochen mehr oder weniger deutlich das Ende von Flash verkündet und die Benutzer zum Umstieg auf HTML5 und Javascript aufgefordert hat, zieht Oracle nun nach und kündigt das Ende von Java Applets an. In Java 9 sollen sie “deprecated”, in einer späteren Version dann gar nicht mehr unterstützt werden.

Aus meiner Sicht ein begrüßenswerter Schritt; leider wird es noch viele Jahre dauern, bis Java (und Flash) ganz aus dem Web verschwunden sind.
