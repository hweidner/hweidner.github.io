Vergleich von Paketsystem-Tools

Im Wiki von Arch Linux gibt es eine vergleichende Übersicht über die Kommandos verschiedener Paketmanager für Aufgaben wie Paketinstallationen, Updates, Paketsuche etc. Verglichen werden die Tools von Arch (pacman), RedHat (yum), Debian (apt-get, aptitude, dpkg), SUSE (zypper) und Gentoo (emerge).
