Git lernen

Das Versionsverwaltungssystem Git wurde von Linus Torvalds geschrieben, nachdem das vorher für die Kernelentwicklung genutzte Bitkeeper nicht mehr zur Verfügung stand. Es erfreute sich von Anfang an einer großen Beliebtheit im Open Source Umfeld. Dementsprechend gibt es viel Dokumentation dazu online. Hier eine Auswahl.

  • Git - der einfache Einstieg
  • Git Tutorial
  • Git Magic
  • Pro Git
  • Webcast: The Basics of Git and Github

Um Git zu nutzen, kann man auf öffentliche Git Hosting Dienste zurückgreifen oder Dienste wie GitWeb, GitPrep oder GitLab auf einem eigenen Server installieren.
