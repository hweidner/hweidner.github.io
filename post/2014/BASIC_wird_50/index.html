BASIC wird 50

Für Tausende von Computerfans war die Programmiersprache BASIC als fest eingebaute Sprache vieler Heimcomputer der Einstieg in die Programmierung. Bei mir war es zunächst das vergleichsweise spartanische Commodore BASIC V2 auf dem Commodore 64, aber für anspruchsvollere Aufgaben wechselte ich schon bald zu SIMONS’ BASIC, das auch die graphischen Möglichkeiten des C64 unterstützte. Zwei Jahre später wechselte ich vom C64 zum Atari ST und programmierte fortan überwiegend in GFA BASIC. Auch unter Linux kann mit X11 Basic in einem GFA-BASIC-ähnlichen Dialekt programmiert werden.

BASIC ist aber wesentlich älter: gestern vor 50 Jahren wurde die erste Version der Programmiersprache veröffentlicht. Auch wenn BASIC heute gerne belächtelt wird, es hatte einen großen Einfluss auf die Computergeschichte. Die Liste von BASIC Dialekten ist beeindruckend, für nahezu jeden Computer vom Heimcomputer bis zum Großrechner gab es die Auswahl unter mehreren BASIC Dialekten. Herzlichen Glückwusch, BASIC!
