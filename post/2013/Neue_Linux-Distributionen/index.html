Neue Linux-Distributionen

Bei Extreme Tech gibt es einen Artikel 5 Ubuntu Alternatives worth checking out. Vorgestellt werden fünf Ubuntu-Derivate mit unterschiedlichen Schwerpunkten. Vier davon stammen von Ubuntu 12.04 LTS ab.

  • BackBox Linux setz den Schwerpunkt auf Security und Vulnerability Scanning Tools.
  • BioLinux wendet sich an Wissenschaftler und Statistiker, insbesondere dem Thema Datenanalyse in der Biologie.
  • PinguyOS legt den Fokus auf Einfachheit und Ähnlichkeit zu Mac OSX und Windows, was Umsteiger von diesen Betriebssystemen ansprechen soll.
  • Poseidon Linux basiert auf Ubuntu Linux 10.04 LTS und legt den Schwerpunkt auf Wissenschaft und Softwareentwicklung.
  • XBMCbuntu ist eine Multimedia Player Distribution basierend auf Ubuntu und XBMC, die als Live-CD funktioniert, aber natürlich auch installiert werden kann.
