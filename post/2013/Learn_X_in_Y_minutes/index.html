Learn X in Y minutes

Learn X in Y minutes ist eine sehr kompakte Darstellung der Syntax zahlreicher Programmiersprachen. Zu jeder Sprache werden die wichtigsten Kontrollstrukturen, Datentypen, I/O Operationen und Idiome anhand eines kommentierten Source Code Listings dargestellt.

Momentan fehlen noch viele wichtige Sprachen, z.B. Perl, Tcl oder Go. Man kann aber selber aktiv werden und einen Pull Request über Github einstellen.

Nachtrag (2013-09-11): Mittlerweile gibt es auch Einträge für Perl und Go.
