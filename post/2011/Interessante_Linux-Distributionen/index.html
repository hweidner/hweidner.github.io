Interessante Linux-Distributionen

Überblick über interessante Entwicklungen aus dem Bereich Linux-Distributionen

Lange Zeit war der Distributionsmarkt zwischen wenigen großen Playern aufgeteilt. Auf Servern, aber auch vielen Desktop-Systemen findet man beiden großen Platzhirsche Debian und RedHat bzw. deren Community-Varianten Fedora und CentOS. Auf Desktops sind OpenSuse und Ubuntu mitsamt seiner Ableger {K,X,L}ubuntu weit verbreitet. Daneben sind Slackware sowie Gentoo bekannt und etabliert.

In der letzten Zeit nimmt (gefühlt) die Vielfalt wieder zu. Hier eine kleine Übersicht über interessante, jüngere Entwicklungen:

  • LinuxMint ist eine Desktop-orientierte Distribution, die sich durch besonderes Augenmerk auf Multimedia-Anwendungen auszeichnet. So sind auch unfreie Multimedia-Codecs und Erweiterungen wie der Flashplayer Bestandteil der Distribution, so dass ein frisch installiertes System sofort Content in den verbreiteten Formaten abspielen kann. Es gibt zwei Varianten: Das ursprüngliche LinuxMint basiert auf Ubuntu Linux und erscheint regelmäßig als Release. MintDebian basiert dagegen auf Debian Testing und ist ein Rolling Release.
  • Arch Linux ist ebenfalls eine Distribution mit Rolling Release. Es wendet sich explizit an erfahrene Anwender und ist für Desktop und Server geeignet. Arch Linux benutzt ein eigenes Paketformat und die Paketverwaltung Pacman.
  • Sabayon wendet sich vorwiegend an Desktop User. Es ist eine Rolling Release Distribution und legt sein Augenmerk auf umfassende Hardwareunterstützung und -Erkennung sowie graphische Effekte. Die Sabayon-Entwicklung basiert auf Gentoo Linux; der Anwender merkt davon allerdings nicht viel, er erhält Binärpakete und installiert mundgerecht von einer Live-DVD.
  • PCLinuxOS war ursprünglich ein Fork von Mandriva , hat sich aber davon mittlerweile weiter weg entwickelt. Es richtet sich an Desktop-Anwender und bietet eine gute Hardwareunterstütztung. Daneben liegt der Fokus auf Einfachheit. PCLinuxOS nutzt den KDE Desktop. Die Paketverwaltung ist RPM, allerdings werden die von Debian bekannten Tools APT und Synaptic anstelle der im RPM-Umfeld verbreiteten Tools genutzt.
  • ZevenOS basiert auf Debian Testing. Es gibt eine Light-Version mit LXDE, die für sehr einfache Rechner ab 600 MHz CPU und 160 MB RAM geeignet ist, und eine Full-Version mit KDE 4. Neben der Unterstützung älterer Rechner liegt der Schwerpunkt auf einfacher Bedienung, Multimedia-Fähigkeiten und Aktualität.
  • SempliceLinux basiert auf Debian und dem schlanken Windowmanager Openbox. Dadurch eignet es sich auch für ältere Hardware. Die Zielgruppe sind Einsteiger, die durch graphische Konfigurationstools bei der Konfiguration unterstützt werden. Dank der Debian-Basis steht eine große Softwareauswahl zur Verfügung.
