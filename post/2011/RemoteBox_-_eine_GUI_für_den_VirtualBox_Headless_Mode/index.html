RemoteBox - eine GUI für den VirtualBox Headless Mode

VirtualBox von Innotec (später SUN und mittlerweile Oracle) ist eine der beliebtesten Virtualisierer für Desktops. Es eignet sich auch gut für Server VMs, die im Headless Mode betrieben werden. Allerdings muss man dann auf die mitgelieferte GUI verzichten und CLI Tools verwenden.

Mit RemoteBox gibt es nun auch eine GUI für den Headless Mode. Die in Perl/Gtk geschriebene Software kommuniziert über die SOAP API mit dem VirtualBox Webservice, wahlweise auf der selben Maschine oder übers Netz. Das Tool hat noch ein paar Haken und Ösen, aber sieht sehr vielversprechend aus. Schlipskompatibel ist das ganze noch nicht, dazu fehlt die Sicht auf mehrere Hosts und Teleportation (aka Live Migration).
