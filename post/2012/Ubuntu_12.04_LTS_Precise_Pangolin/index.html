Ubuntu 12.04 LTS Precise Pangolin

Als ich noch Ubuntu auf dem Desktop genutzt habe, habe ich nur ein einziges Mal eine neue Version auf dem produktiven Arbeitsplatzrechner installiert, bevor sie offiziell freigegeben wurde. Das war Ubuntu 12.04 LTS Precise Pangolin. Hier einige Anmerkungen in Bezug auf den Installationsbericht zur Version 11.10.

  • Unity ist weiterhin der Standard-Desktop, es stehen aber die Gnome 3 Shell und der Gnome 3 Classic Mode zur Auswahl. Beides funktioniert mit und ohne 3D-Unterstützung seitens der Grafikkarte. Auch Unity funktioniert mittlerweile in einem VirtualBox Gastsystem.

  • Der Kernel-Bug im Zusammenhang mit einem Twinhan DVB-T Receiver ist behoben. Allerdings zickt der Infrarot-Fernbedienungs-Empfänger: wenn man eine Fernsehsendung mit DVB-T laufen hat und parallel in einem anderen Fenster arbeitet, wird dort permanent das Drücken der TAB-Taste emuliert. Um den DVB-T Stick vernünftig zu nutzen, muss die Fernbedienung abgeschaltet werden. Dazu erstellt man eine Datei /etc/modprobe.d/dvbt.conf mit folgenden Inhalt:

    options dvb-usb disable_rc_polling=1

  • Rhythmbox ist wieder das Standard-Musikwiedergabeprogramm. Die Abstürze von 11.10 treten bei Ubuntu 12.04 LTS nicht mehr auf.

Nachtrag: Heute, am 26.4.2012, wurde Ubunu 12.04 LTS offiziell freigegeben (siehe auch Pro-Linux, Heise und Golem)
