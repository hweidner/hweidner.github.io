Vergleich von Skriptsprachen

Vergleich von Konzepten und Syntax der Skriptsprachen PHP, Perl, Python und Ruby

Unter http://hyperpolyglot.org/scripting gibt es einen übersichtlichen Vergleich der vier Skriptsprachen PHP, Perl, Python und Ruby. Verglichen werden die Syntax, Kontrollstrukturen, Funktionen für Arithmetik und Stringverarbeitung, Systemprogrammierung und I/O Operationen.
